## üö® Critical Behavior Rules

1.  **Never Run Interactive Init:** The command `npx prisma init --db` requires user input (Region/Project). You must **ask the user** to run it.
2.  **Never Invent URLs:** The `DATABASE_URL` is generated by the CLI. Do not write a placeholder like `postgres://user:pass@localhost`.
3.  **Never Skip Dotenv:** `prisma.config.ts` MUST start with `import "dotenv/config"`.
